# 文本匹配算法
### 说明
文本匹配算法，包含基于表示和基于交互的两种基础架构。  
基于表示：DSSM  
基于交互：ARC-II、ESIM、Bert
### 文件结构
```
Text_Matching
├── DSSM
├── ARC-II
├── ESIM
├── bert
└── 数据准备
```
### 结果对比
| 模型 | loss | acc | 
| ---- | ---- | ---- | 
| DSSM | 0.4326 | 0.8039|  
| ARC-II | 0.4765 | 0.8022|  
| ESIM | 0.3943| 0.829 | 
| bert | 0.8041| 0.8444 | 

目前结果只是针对示例数据集进行了简单调参的结果，实际应用还需自行调整超参。
