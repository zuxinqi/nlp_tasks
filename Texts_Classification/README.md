# 文本分类算法
### 说明
包含六种基础算法，基本都适用于短文本分类。机器学习文本分类、fasttext、han可用于长文本分类。在示例数据集中，bert效果最好，fasttext训练速度较快，效果也较好。
### 文件结构
```
Texts_Classification
├── 机器学习_文本分类
├── fasttext_文本分类
├── textcnn_文本分类
├── lstm_文本分类
├── han_文本分类
├── dgcnn_文本分类
├── dpcnn_文本分类
├── rcnn_文本分类
├── bert_文本分类
└── 数据准备
```
